Repeat Landscape Generator

This is a series of scripts to create stunning visualizations from repeats identified by RepeatMasker based on Kimura divergence (excluding CpG), the *.align.out file is parsed and converted to a *.csv file, which can be plotted in stacked-histogram or color-full pies with R or Google Charts API.

*Work in progress, please contact me if you are using this.
